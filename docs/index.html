Seznam tříd

JAX

JAX je jmenný prostor a také funkce, která najde element, který odpovídá selectoru nebo obalí zadaný objekt vlastní třídou (wrapperem)

JAX.Async

JAX.Async je třída, která umožňuje zpracovat sekvenci nejen Promisů, ale i animací pomocí JAX.FX.

JAX.Document

JAX.Document je třída reprezentující instanci window.Document.

JAX.DocumentFragment

JAX.DocumentFragment je třída reprezentující instanci window.DocumentFragment.

JAX.Element

JAX.Element je třída reprezentující HTML Element.

JAX.Event

JAX.Event je třída obalující window.Event pro snadnější práci se stavem událostí

JAX.FX

JAX.FX je pomocník pro snadnější tvorbu animací

JAX.FX.CSS3

JAX.FX.CSS3 je pomocník pro animaci pomocí CSS3 transitions

JAX.FX.Interpolator

JAX.FX.Interpolator je pomocník pro animaci pomocí interpolátoru

JAX.FX.Scrolling

JAX.FX.Scrolling je pomocník pro animaci scrollování

JAX.FXArray

JAX.FXArray je třída rezrezentující pole JAX.FX instancí.

JAX.IAnimateableNode

JAX.IAnimateableNode tvoří rozhraní pro nody, které lze animovat

JAX.IIterable

JAX.IIterable tvoří rozhraní pro iterovatelé prvky, které je svým chováním velice podobné datovému typu Array

JAX.IListening

JAX.IListening tvoří rozhrani implementujicí práci s navěšováním a odvěšováním událostí

JAX.IMoveableNode

JAX.IMoveableNode tvoří rozhraní pro nody, kterými jde manipulovat v rámci DOMu

JAX.INodeWithChildren

JAX.INodeWithChildren tvoří rozhraní pro nody, které mohou mít potomky

JAX.IScrollableNode

JAX.IScrollableNode tvoří rozhraní pro nody, u který lze scrollovat obsahem

JAX.ISearchableNode

JAX.ISearchableNode tvoří rozhraní pro nody, které lze prohledávat pomocí querySelector a querySelectorAll

JAX.Listener

JAX.Listener je třída rezrezentující posluchač události

JAX.ListenerArray

JAX.ListenerArray je třída rezrezentující pole posluchačů.

JAX.Node

JAX.Node je obecná třída reprezentující základ JAXovských elementů

JAX.NodeArray

JAX.NodeArray je třída reprezentující pole instancí JAX.Node a poskytující metody pro hromadné zpracování.

JAX.NullNode

JAX.NullNode je třída reprezentující nullový node - návrhový vzor Null object.

JAX.TextNode

JAX.TextNode je třída reprezentující text node a comment node (elm.nodeType == 3 || elm.nodeType == 8).

JAX.Window

JAX.Window je třída reprezentující object Window.
