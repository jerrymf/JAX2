Třída JAX.FX

JAX.FX je pomocník pro snadnější tvorbu animací
Definováno v: jax.js

Metody - souhrn
Atributy metody Jméno a popis metody
addProperty(property, duration, start, end, method)
Přidá css vlastnost, která se bude animovat. Pro každou vlastnost lze zadat různou délku animace a také hodnoty, od kterých se má začít a po které skončit.
addTranslateProperty(duration, start, end, method)
Přidá transformační vlastnost (translateX, translateY, translateZ). Používá fallback pro prohlížeče, které transformace neumí a to přes elm.style.top a elm.style.left.
isRunning()

reverse()

run()

stop()

funkce, která se zavolá, jakmile animace skončí. V případě prvního parametru se jedná o úspěšné dokončení, v případě druhého o chybu.

{JAX.FX} addProperty(property, duration, start, end, method)
Přidá css vlastnost, která se bude animovat. Pro každou vlastnost lze zadat různou délku animace a také hodnoty, od kterých se má začít a po které skončit.
Parametry:
{string} property
css vlastnost, která se má animovat
{string || number} duration
délka animace - lze zadat i jednotky s nebo ms (výchozí jsou ms)
{string || number} start
počáteční hodnota - je dobré k ní uvést vždy i jednotky (pokud jde o číselnou hodnotu) a jako výchozí se používají px
{string || number} end
koncová hodnota - je dobré k ní uvést vždy i jednotky (pokud jde o číselnou hodnotu) a jako výchozí se používají px
{string} method
css transformační metoda (ease, linear, ease-in, ease-out, ... ) více na webu W3C, pozn.: pokud prohlížeč neumí transitions, je použito js řešení a metoda je vždy LINEAR
Vrací:
{JAX.FX}

Přidá transformační vlastnost (translateX, translateY, translateZ). Používá fallback pro prohlížeče, které transformace neumí a to přes elm.style.top a elm.style.left.

{object} end

{number || string} end.z

{string} method
