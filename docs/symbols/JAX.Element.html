Třída JAX.Element

JAX.Element je třída reprezentující HTML Element. Implementuje následující rozhraní: JAX.INodeWithChildren, JAX.IMoveableNode, JAX.ISearchableNode, JAX.IListening, JAX.IAnimateableNode, JAX.IScrollableNode
Definováno v: jax.js
